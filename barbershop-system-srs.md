## 1. วัตถุประสงค์
เอกสารฉบับนี้จัดทำขึ้นเพื่อกำหนดขอบเขตการดำเนินงาน เงื่อนไข และข้อตกลงในการพัฒนาระบบจองห้องพักออนไลน์ ระหว่างผู้ว่าจ้างและผู้รับจ้าง เพื่อให้การดำเนินโครงการเป็นไปอย่างมีประสิทธิภาพและบรรลุวัตถุประสงค์ที่กำหนดไว้
- จัดการการจองคิวตัดผม
- บริหารจัดการช่างและตารางการทำงาน
- จัดเก็บข้อมูลลูกค้าและประวัติทรงผม
- จัดการบริการและราคา
- ควบคุมการชำระเงินและการเงิน
- จัดการโปรโมชันและสมาชิก
- สร้างรายงานและวิเคราะห์ข้อมูล

## 2. ความต้องการด้านฟังก์ชันการทำงาน
### 2.1 การจัดการลูกค้าและสมาชิก
#### 2.1.1 การจัดการข้อมูลลูกค้า (Customer information)


| Requirement ID | Requirement Description | Priority |
|---------------|------------------------|----------|
| CUS-01 | ข้อมูลลูกค้า: รหัสลูกค้า (สร้างอัตโนมัติ) ชื่อ-นามสกุล เพศ เบอร์โทรศัพท์ อีเมล Line ID วันเกิด รูปถ่าย วันที่สมัครสมาชิก หมายเหตุพิเศษ | High |
| CUS-02 | ประวัติทรงผม: รูปถ่ายทรงผมแต่ละครั้ง รายละเอียดการตัด ช่างที่ตัด วันที่ตัด ราคา ความพึงพอใจ | Medium |
| CUS-03 | การจัดการข้อมูล: เพิ่มลูกค้าใหม่ แก้ไขข้อมูลลูกค้า ค้นหาลูกค้า (ชื่อ/เบอร์โทร) ดูประวัติการใช้บริการ ระงับ/เปิดใช้งานบัญชี | High |


#### 2.1.2 ระบบสมาชิก (User Management)


| Requirement ID | Requirement Description | Priority |
|---------------|------------------------|----------|
| USR-01 | ข้อมูลสมาชิก: ระดับสมาชิก (Bronze, Silver, Gold, Platinum) แต้มสะสม ส่วนลดพิเศษ วันหมดอายุสมาชิก สิทธิพิเศษ | High |
| USR-02 | การสะสมแต้ม: 1 บาท = 1 แต้ม แต้มพิเศษในวันเกิด (x2) แต้มพิเศษตามระดับสมาชิก แลกแต้มเป็นส่วนลด แต้มหมดอายุ 1 ปี | Medium |
| USR-03 | สิทธิประโยชน์: ส่วนลดตามระดับ (5-20%) จองคิวล่วงหน้าได้ ข้ามคิวได้ (Platinum) ของขวัญวันเกิด โปรโมชันพิเศษ | High |
